# 방문자 집계 켜기 (GoatCounter, 3분·무료)

베이비블룸에 몇 명이 들어오는지 보려면 GoatCounter를 켜면 돼요.
개인정보를 수집하지 않고(쿠키·IP 저장 안 함) 방문 수만 세는 도구라 육아앱에 적합합니다.

## 1. 가입 & 사이트 코드 만들기
1. https://www.goatcounter.com 접속 → **Sign up**
2. **Code** 칸에 원하는 이름 입력 (예: `babybloom`)
   → 대시보드 주소가 `https://babybloom.goatcounter.com` 이 됩니다.
3. 이메일 인증하고 로그인

## 2. 앱에 코드 넣기
`js/config.js`를 열어 방금 정한 코드를 넣으세요:

```js
const GOATCOUNTER_CODE = 'babybloom';   // ← 1번에서 정한 Code
```

## 3. 배포
`sw.js`의 `VERSION`을 한 단계 올리고(예: v15 → v16) 커밋·푸시하면 끝.
저한테 "집계 코드 넣었어"라고 하시면 배포까지 해드릴게요.

## 4. 확인
`https://<코드>.goatcounter.com` 에 로그인하면
- 오늘/이번 주/전체 방문자 수
- 어떤 페이지를 봤는지, 어디서 유입됐는지(쓰레드·유튜브 등)
- 기기·브라우저·국가
를 볼 수 있어요.

## 참고
- **과거 방문자는 소급 집계되지 않아요.** 코드를 켠 시점부터 셉니다.
- 홈 화면에 설치(PWA)해서 여는 방문도 집계돼요.
- 개인정보를 안 모으기 때문에 "동의 배너"가 필요 없습니다.
- 더 상세한 지표(체류시간·전환 등)가 필요해지면 나중에 GA4로 바꿀 수 있어요.
