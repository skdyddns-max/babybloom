# 가족 공유 설정 방법 (5분, 무료)

가족 공유는 무료 Supabase 데이터베이스를 사용해요. 한 번만 설정하면 됩니다.

## 1. Supabase 프로젝트 만들기
1. https://supabase.com 접속 → 무료 가입 → **New Project**
2. 이름 아무거나(예: babybloom), 리전 **Northeast Asia (Seoul)**, DB 비밀번호 저장

## 2. 테이블 만들기
프로젝트 → 왼쪽 **SQL Editor** → 아래 그대로 붙여넣고 **Run**:

```sql
create table families (
  code text primary key,
  data jsonb not null,
  updated_at timestamptz not null default now()
);

alter table families enable row level security;

create policy "family anon access" on families
  for all using (true) with check (true);
```

## 3. 키 복사해서 앱에 넣기
1. 프로젝트 → **Project Settings → API**
2. **Project URL**과 **anon public** 키 복사
3. 이 저장소의 `js/config.js`를 열어 두 값을 붙여넣기:

```js
const SUPABASE_CONFIG = {
  url: 'https://xxxx.supabase.co',
  anonKey: 'eyJhbGciOi...',
};
```

4. `sw.js`의 `VERSION`을 한 단계 올리고(예: v4 → v5) 커밋·푸시

## 4. 사용법
- 동생 폰: 아기 정보 수정 → 가족 공유 → **가족 코드 만들기** → 코드 6자리를 남편에게 전송
- 남편 폰: 같은 화면에서 코드 입력 → **참여** → 끝. 이후 기록이 자동으로 합쳐져요(변경 2.5초 후 업로드, 60초마다·화면 복귀 시 내려받기)

## 참고
- anon 키는 공개되어도 되는 키지만, **코드를 아는 사람은 그 가족 데이터를 볼 수 있는 구조**예요(가족용 간이 공유). 이름·기록 외 민감정보는 저장되지 않아요.
- 기록 병합은 합집합이라 두 사람이 동시에 기록해도 유실되지 않아요. 삭제는 상대 기기가 아직 안 받았으면 되살아날 수 있어요(한계).
